# Software



## Requirements
- [Node](https://nodejs.org) 
- [Bun](https://bun.sh/)
- sshpass (only if you want to deploy on raspberry pi with `bun run deploy` command)

###  Setup
```bash
$ bun install
$ bun test # check if everything is ok
$ bun dev # Hot reload at localhost:3000  
```
